This cross-\/platform C++ library provides its consumers the ability to compile and execute Arduino sketches on a hosted environment, with bindings to its virtual I/O ports to allow the host application to interact with its child sketches.

Check our \href{https://github.com/ItJustWorksTM/libSMCE/wiki}{\texttt{ Wiki}} for more information about usage.\hypertarget{md__r_e_a_d_m_e_autotoc_md17}{}\doxysubsubsection{Supported I/O}\label{md__r_e_a_d_m_e_autotoc_md17}

\begin{DoxyItemize}
\item GPIO (Arduino\textquotesingle{}s {\ttfamily digital\+Read}, {\ttfamily analog\+Write}, etc)
\item UART (Arduino\textquotesingle{}s {\ttfamily Serial} global)
\item SD
\item MQTT (interface of \href{https://github.com/256dpi/arduino-mqtt}{\texttt{ arduino-\/mqtt}}) -\/ Note\+: cannot be monitored by host application
\item OV767X Camera (interface of \href{https://github.com/arduino-libraries/Arduino_OV767X}{\texttt{ Arduino\+\_\+\+OV767X}}) -\/ Note\+: available pixel formats differ
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md18}{}\doxysubsection{Build Requirements}\label{md__r_e_a_d_m_e_autotoc_md18}
